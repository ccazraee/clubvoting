
index.html - to point to govote.shtml or closevote.shtml
govote.shtml - main file for voting. (list of candidates is here)
closevote.shtml - file to inform that voting has already closed
thanks.shtml - thank you for voting message file

voting.js - main javascript file for govote.shtml
result.php - process sanitized.txt and convert as result
process.php - accept form data and insert records into logincheck.txt and voting.txt

vote.txt - raw voting file
sanitized.txt - cleaned up and sorted voting file
logincheck.txt - record login history (extract of form data from vote.txt)
username.txt - list of staff eligible to vote
